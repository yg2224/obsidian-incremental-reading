# Obsidian Incremental Reading Plugin

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Obsidian](https://img.shields.io/badge/Obsidian-Plugin-7C3AED.svg)](https://obsidian.md)

智能增量阅读插件 - 基于智能算法的个性化学习系统，帮助你在 Obsidian 知识库中高效地进行增量阅读和知识发现。

**让您的知识库主动为您推荐下一篇该读什么**

## 📖 简介

Obsidian Incremental Reading 插件将您的知识库转化为一个智能学习系统。通过先进的推荐算法和多维度优先级管理，帮助您在成百上千的笔记中找到最值得阅读的文档，实现真正的增量阅读体验。

不同于传统的随机漫游，本插件提供三种智能漫游模式，基于文档的内容相似度、优先级权重和访问历史，为您推荐最合适的学习材料。

## ✨ 核心特性

### 🎯 三种智能漫游模式

#### 1. 继续漫游 (Continue Roaming)
**基于优先级的加权随机选择**

- 综合考虑文档的难度、重要性、紧急度、兴趣度和手动优先级
- 根据综合优先级计算权重，优先级越高，选中概率越大
- 实时显示每个文档的选中概率和优先级分数
- 排除当前正在查看的文档，避免重复

#### 2. 智能推荐 (Smart Recommendations)
**基于 TF-IDF 的内容相似度推荐**

- 使用您的阅读历史作为锚点文档
- 通过 TF-IDF 算法分析文档内容相似度
- 计算余弦相似度，推荐相关内容
- 支持中英文混合分词

#### 3. 随机漫游 (Random Roaming)
**纯随机选择**

- 从漫游列表中完全随机选择文档
- 避免算法偏见，发现意外之喜
- 适合探索式学习和打破思维定式

### 📊 五维文档评估体系

每个文档都有 5 个核心指标（0-10 分）：

| 指标 | 说明 | 默认权重 |
|------|------|---------|
| **难度 (Difficulty)** | 内容理解难度 | 2.0 |
| **重要性 (Importance)** | 知识的重要程度 | 3.0 |
| **紧急度 (Urgency)** | 阅读的紧急程度 | 2.5 |
| **兴趣度 (Interest)** | 个人兴趣偏好 | 2.0 |
| **手动优先级 (Priority)** | 手动设置的优先级权重 | 2.5 |

优先级范围约为 25-125（取决于权重配置），系统会根据此优先级计算选中概率。

## 🚀 快速开始

### 安装方法

1. 复制到 Obsidian 插件目录：`<vault>/.obsidian/plugins/obsidian-incremental-reading/`
2. 重启 Obsidian 并在设置中启用插件

### 基础使用

1. **添加文档到漫游列表**：打开文档，点击"加入漫游"
2. **设置文档指标**：调整难度、重要性、紧急度、兴趣度滑块
3. **开始漫游**：点击"继续漫游"或"智能推荐"
4. **查看概率**：界面显示每个文档的选中概率

## ⚙️ 技术实现

### 优先级计算

```
优先级 = Σ(指标 × 权重)

其中：
- 难度 × 难度权重 (默认 2.0)
- 重要性 × 重要性权重 (默认 3.0)
- 紧急度 × 紧急度权重 (默认 2.5)
- 兴趣度 × 兴趣度权重 (默认 2.0)
- 手动优先级 × 优先级权重 (默认 2.5)
```

### 加权随机选择

```typescript
// 计算权重
weight = max(0.1, 优先级)

// 选择概率
选择概率 = (文档权重 / 总权重) × 100%
```

选中概率与优先级成正比，优先级越高的文档越容易被选中。

### 智能推荐算法

```
综合评分 = 相似度 × 0.4 + 优先级 × 0.3 + 时效性 × 0.2 + 访问频率 × 0.1
```
