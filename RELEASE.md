# Incremental Reading v2.0.0 - 完全中文化智能增量阅读系统

## 🎯 版本亮点

### 🇨🇳 完整中文化界面
- **移除i18n系统**：告别复杂的国际化配置
- **原生中文界面**：所有文本直接嵌入，显示稳定可靠
- **优化用户体验**：不再有翻译键名显示问题

### 📊 智能指标管理系统
- **自定义指标**：支持1-10个自定义评估指标
- **自动权重归一化**：确保评分系统的一致性
- **实时更新**：指标修改后立即同步到插件界面
- **直观操作**：拖拽调整权重，滑块控制数值

### 🧠 智能推荐算法
- **TF-IDF相似度分析**：基于文档内容智能推荐
- **多重推荐基准**：结合最近浏览和高频访问文档
- **个性化推荐**：根据您的阅读习惯定制推荐结果
- **推荐质量优化**：可调节候选文档数量和采样段落

### 📁 批量文件管理
- **文件夹批量添加**：一键添加整个文件夹的文档
- **多选文件支持**：Ctrl+点击选择多个文档
- **智能过滤**：自动排除非Markdown文件和特定路径
- **灵活配置**：支持通配符路径排除

### ⚡ 性能优化
- **容器隔离机制**：设置组件互不影响，界面更稳定
- **增量更新**：只刷新变化的部分，提升响应速度
- **内存优化**：高效的文档索引和缓存机制

## 📋 安装说明

### 🚀 快速安装
1. 下载 `main.js`、`manifest.json`、`styles.css` 三个文件
2. 在Obsidian中打开 `设置 > 社区插件 > 浏览`
3. 关闭安全模式（如果开启）
4. 点击"从本地安装插件"
5. 选择插件文件所在的文件夹

### ⚙️ 系统要求
- **Obsidian版本**: 0.15.0 或更高
- **平台支持**: Windows、macOS、Linux
- **内存建议**: 至少 512MB 可用内存

## 📖 使用指南

### 🎯 快速上手
1. **添加文档**：使用"加入漫游"功能添加想要阅读的文档
2. **设置指标**：在设置中配置您关心的评估维度
3. **开始阅读**：点击"继续漫游"开始智能增量阅读
4. **查看推荐**：根据智能推荐发现相关文档

### ⚡ 高级技巧
- **批量操作**：使用文件夹批量添加提高效率
- **自定义权重**：根据学习重点调整指标权重
- **定期清理**：使用数据管理功能维护插件性能
